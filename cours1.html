Bienvenue au cours

Ce cours est conçu pour introduire les concepts fondamentaux du développement Web. Vous apprendrez les bases du HTML, du CSS et du JavaScript, ainsi que les bonnes pratiques pour créer des sites Web modernes et accessibles.

Objectifs du cours

  • Comprendre la structure d'une page Web avec HTML
  • Apprendre à styliser une page avec CSS
  • Découvrir les bases de l'interactivité avec JavaScript
